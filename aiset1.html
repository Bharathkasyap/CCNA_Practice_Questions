Q1: Which IPv4 address class provides up to 254 usable host addresses per network?

  • A. Class A
  • B. Class B
  • C. Class C
  • D. Class D

Answer: C. Class C – supports 254 usable hosts ( /24 network).

Q2: Which routing protocol uses DUAL (Diffusing Update Algorithm)?

  • A. RIP
  • B. EIGRP
  • C. OSPF
  • D. BGP

Answer: B. EIGRP.

Q3: Which layer of the OSI model ensures reliable delivery and flow control?

  • A. Network

  • D. Application

Answer: B. Transport layer (TCP provides reliability and flow control).
