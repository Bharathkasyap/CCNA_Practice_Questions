D NAT PAT and ACL

  1. Configure PAT to outside interface. Name the three parts and one show command that proves translations.
  2. Block Telnet from 10.0.2.0 to 10.0.1.0 inbound on G0 0 then permit all. Give order and attachment and one show command for hits.
  3. NAT not translating because ip nat outside is missing. State where to add it and one show and one debug to confirm.
Show answers

Inside and outside on interfaces, ACL for inside source, ip nat inside source list 1 interface G0 1 overload, verify with show ip nat translations

access-list 101 deny tcp 10.0.2.0 0.0.0.255 10.0.1.0 0.0.0.255 eq 23 then permit ip any any, apply ip access-group 101 in on G0 0, verify with show access-lists

Add ip nat outside on G0 1, show ip nat statistics and debug ip nat

F Wireless WPA2 PSK

  1. Create SSID CCNA LAB with WPA2 PSK Cisco123 on autonomous AP and name one show command to verify SSID is active.
  2. Clients cannot join because radio is down. Provide the command to bring it up and the show command that confirms channel and power.
Show answers

dot11 ssid CCNA_LAB with authentication open and authentication key-management wpa2 and wpa-psk ascii Cisco123 then bind on Dot11Radio. Verify with show dot11 associations or show running-config | section ssid

no shutdown on radio interface. show interface Dot11Radio or show controllers dot11radio
