CCNA Practice Set 2

Q1: What is the maximum number of usable host addresses in a /25 subnet?

  • A. 62
  • B. 126
  • C. 128
  • D. 254

Answer: A. 62 usable hosts.

Q2: Which Cisco command displays the current IOS version?

  • A. show interfaces
  • B. show version
  • C. show running-config
  • D. show flash

Answer: B. show version.

Q3: Which VLAN range is reserved and cannot be used for normal VLANs?

  • A. 1–1005
  • B. 1006–4094
  • C. 0 and 4095
  • D. 500–600

Answer: C. VLAN 0 and 4095 are reserved.

Q4: Which OSPF area type blocks external routes but allows inter-area routes?

  • A. Standard area
  • B. Stub area
  • C. NSSA
  • D. Backbone area

Answer: B. Stub area.

Q5: Which two protocols are used by IP phones to communicate with a Cisco Call Manager? (Choose two)

  • A. SCCP
  • B. SIP
  • C. RTP
  • D. H.323

Answer: A. SCCP, B. SIP.

Q6: Which routing protocol uses AS-path as its main metric?

  • A. RIP
  • B. OSPF
  • C. BGP
  • D. EIGRP

Answer: C. BGP.

Q7: Which Cisco command saves the running configuration to NVRAM?

  • A. copy run tftp
  • B. write erase
  • C. copy running-config startup-config
  • D. reload

Answer: C. copy running-config startup-config.

Q8: Which command displays CPU and memory utilization?

  • A. show processes cpu
  • B. show memory
  • C. show processes
  • D. show system

Answer: A. show processes cpu.

Q9: What is the default administrative distance of OSPF?

  • A. 90
  • B. 100
  • C. 110
  • D. 120

Answer: C. 110.

Q10: Which spanning-tree mode supports rapid convergence?

  • A. PVST+
  • B. RSTP (802.1w)
  • C. STP (802.1D)
  • D. MST

Answer: B. RSTP.

Q11: Which IPv6 feature allows automatic configuration of addresses without DHCP?

  • A. SLAAC
  • B. NDP
  • C. EUI-64
  • D. All of the above

Answer: D. SLAAC uses NDP and EUI-64 for automatic address assignment.

Q12: Which cable type is required to connect two Cisco switches directly?

  • A. Crossover cable
  • B. Straight-through cable
  • C. Rollover cable
  • D. Coaxial cable

Answer: A. Crossover cable (if auto-MDI/MDIX is not enabled).

Q13: Which two message types are used in DHCP? (Choose two)

  • A. Discover
  • B. Offer
  • C. Request
  • D. Update

Answer: A. Discover, B. Offer (part of DORA process).

Q14: Which Cisco technology provides IP-to-MAC mapping security against spoofing?

  • A. ARP
  • B. DAI
  • C. Port security
  • D. DHCP snooping

Answer: B. Dynamic ARP Inspection.

Q15: Which two protocols are considered distance-vector? (Choose two)

  • A. RIP
  • B. BGP
  • C. OSPF
  • D. EIGRP

Answer: A. RIP, D. EIGRP (hybrid but distance-vector characteristics).

Q16: Which wireless standard supports speeds up to 600 Mbps on 2.4 GHz and 5 GHz?

  • A. 802.11a
  • B. 802.11b
  • C. 802.11g
  • D. 802.11n

Answer: D. 802.11n.

Q17: Which Cisco command shows the VTP domain name and mode?

  • A. show vtp status
  • B. show vlan brief
  • C. show spanning-tree
  • D. show interface trunk

Answer: A. show vtp status.

Q18: Which routing protocol is classified as an EGP (Exterior Gateway Protocol)?

  • A. RIP
  • B. OSPF
  • C. EIGRP
  • D. BGP

Answer: D. BGP.

Q19: Which Cisco command disables CDP globally?

  • A. no cdp run
  • B. no cdp enable
  • C. disable cdp
  • D. shutdown cdp

Answer: A. no cdp run.

Q20: Which IP address is reserved for the local broadcast?

  • A. 255.255.255.255
  • B. 127.0.0.1
  • C. 224.0.0.1
  • D. 0.0.0.0

Answer: A. 255.255.255.255.

Q21: Which OSPF router type connects different areas?

  • A. ABR
  • B. ASBR
  • C. DR
  • D. BDR

Answer: A. Area Border Router (ABR).

Q22: Which Cisco command shows IP SLA statistics?

  • A. show ip sla configuration
  • B. show ip sla statistics
  • C. show ip route
  • D. show sla summary

Answer: B. show ip sla statistics.

Q23: Which two types of cables are used for serial WAN connections? (Choose two)

  • A. V.35
  • B. RS-232
  • C. Fiber
  • D. HDMI

Answer: A. V.35, B. RS-232.

Q24: Which command shows detailed EIGRP neighbor information?

  • A. show ip eigrp neighbors
  • B. show ip eigrp topology
  • C. show ip route eigrp
  • D. show ip eigrp traffic

Answer: A. show ip eigrp neighbors.

Q25: Which type of NAT allows multiple private IPs to map to one public IP using ports?

  • A. Static NAT
  • B. Dynamic NAT
  • C. PAT
  • D. NAT64

Answer: C. PAT (Port Address Translation).

Q26: Which Cisco technology allows grouping of multiple physical links into one logical link?

  • A. Port Security
  • B. EtherChannel
  • C. StackWise
  • D. HSRP

Answer: B. EtherChannel.

Q27: What is the default administrative distance of static routes with next-hop IP?

  • A. 0
  • B. 1
  • C. 90
  • D. 120

Answer: B. 1.

Q28: Which Cisco command shows trunking encapsulation type on an interface?

  • A. show interfaces trunk
  • B. show vlan brief
  • C. show running-config
  • D. show spanning-tree

Answer: A. show interfaces trunk.

Q29: Which command enables IPv6 routing on Cisco IOS devices?

  • A. enable ipv6
  • B. ipv6 unicast-routing
  • C. router ipv6
  • D. ipv6 enable

Answer: B. ipv6 unicast-routing.

Q30: Which two authentication methods are supported by PPP? (Choose two)

  • A. PAP
  • B. CHAP
  • C. Kerberos
  • D. RADIUS

Answer: A. PAP, B. CHAP.

Q31: Which protocol is used to resolve hostnames to IP addresses?

  • A. DHCP
  • B. ARP
  • C. DNS
  • D. ICMP

Answer: C. DNS.

Q32: Which IPv6 address type is equivalent to IPv4 private addresses?

  • A. Global Unicast
  • B. Anycast
  • C. Unique Local
  • D. Link-Local

Answer: C. Unique Local (fc00::/7).

Q33: Which OSPF router type connects an OSPF domain to other routing domains?

  • A. ABR
  • B. ASBR
  • C. DR
  • D. BDR

Answer: B. ASBR (Autonomous System Boundary Router).

Q34: Which Cisco feature prevents loops by shutting down ports receiving unexpected BPDUs?

  • A. PortFast
  • B. BPDU Guard
  • C. Root Guard
  • D. UDLD

Answer: B. BPDU Guard.

Q35: Which protocol uses port 179?

  • A. OSPF
  • B. RIP
  • C. BGP
  • D. EIGRP

Answer: C. BGP.

Q36: Which IPsec component provides encryption?

  • A. AH
  • B. ESP
  • C. GRE
  • D. SSL

Answer: B. ESP (Encapsulating Security Payload).

Q37: Which Cisco command shows the MAC address table?

  • A. show mac-address-table
  • B. show arp
  • C. show vlan brief
  • D. show interfaces

Answer: A. show mac-address-table.

Q38: Which command verifies NTP synchronization?

  • A. show clock
  • B. show ntp status
  • C. show time
  • D. show running-config

Answer: B. show ntp status.

Q39: Which two states are valid STP port states? (Choose two)

  • A. Blocking
  • B. Listening
  • C. Reset
  • D. Shutdown

Answer: A. Blocking, B. Listening.

Q40: Which Cisco command shows DHCP bindings?

  • A. show dhcp leases
  • B. show ip dhcp binding
  • C. show running-config
  • D. show ip interface brief

Answer: B. show ip dhcp binding.

Q41: Which IPv6 address block is used for multicast?

  • A. FF00::/8
  • B. FE80::/10
  • C. FC00::/7
  • D. 2000::/3

Answer: A. FF00::/8.

Q42: Which two commands enable port security on a switch? (Choose two)

  • A. switchport mode access
  • B. switchport port-security
  • C. no shutdown
  • D. spanning-tree enable

Answer: A. switchport mode access, B. switchport port-security.

Q43: Which protocol is used by Cisco devices for secure device management?

  • A. Telnet
  • B. SSH
  • C. SNMP
  • D. FTP

Answer: B. SSH.

Q44: Which command verifies EIGRP neighbors?

  • A. show ip eigrp traffic
  • B. show ip eigrp neighbors
  • C. show ip eigrp topology
  • D. show ip route eigrp

Answer: B. show ip eigrp neighbors.

Q45: Which two protocols are considered link-state? (Choose two)

  • A. OSPF
  • B. IS-IS
  • C. RIP
  • D. BGP

Answer: A. OSPF, B. IS-IS.

Q46: Which spanning-tree port role forwards traffic away from the root bridge?

  • A. Root Port
  • B. Designated Port
  • C. Blocking Port
  • D. Alternate Port

Answer: B. Designated Port.

Q47: Which Cisco command shows interface errors?

  • A. show interfaces
  • B. show ip interface brief
  • C. show running-config
  • D. show vlan brief

Answer: A. show interfaces.

Q48: Which NAT type allows a one-to-one mapping between private and public IPs?

  • A. Static NAT
  • B. Dynamic NAT
  • C. PAT
  • D. NAT64

Answer: A. Static NAT.

Q49: Which command enables spanning-tree PortFast on an interface?

  • A. spanning-tree portfast
  • B. switchport portfast
  • C. spanning-tree fast
  • D. spanning-tree enable

Answer: A. spanning-tree portfast.

Q50: Which IP address range is Class B private?

  • A. 10.0.0.0/8
  • B. 172.16.0.0/12
  • C. 192.168.0.0/16
  • D. 224.0.0.0/4

Answer: B. 172.16.0.0 – 172.31.255.255.

Q51: Which Cisco command verifies VLAN membership of an interface?

  • A. show vlan brief
  • B. show vlan id
  • C. show vlan interface
  • D. show interfaces trunk

Answer: A. show vlan brief.

Q52: Which two methods are used to mitigate VLAN hopping attacks? (Choose two)

  • A. Disable unused ports
  • B. Use dynamic trunking
  • C. Configure unused ports as access
  • D. Enable VTP

Answer: A. Disable unused ports, C. Configure as access.

Q53: Which OSPF LSA type is generated by ASBRs?

  • A. Type 1
  • B. Type 3
  • C. Type 5
  • D. Type 7

Answer: C. Type 5 External LSAs.

Q54: Which IPv6 address is equivalent to IPv4 loopback 127.0.0.1?

  • A. ::
  • B. ::1
  • C. fe80::1
  • D. 2000::1

Answer: B. ::1.

Q55: Which Cisco feature allows multiple devices to share the same virtual IP for redundancy?

  • A. STP
  • B. EtherChannel
  • C. HSRP
  • D. PortFast

Answer: C. HSRP.

Q56: Which Cisco command enables trunking on an interface?

  • A. switchport trunk allowed vlan
  • B. switchport mode trunk
  • C. switchport trunk encapsulation dot1q
  • D. switchport mode access

Answer: B. switchport mode trunk.

Q57: Which IPv4 address class has a default subnet mask of 255.0.0.0?

  • A. Class A
  • B. Class B
  • C. Class C
  • D. Class D

Answer: A. Class A.

Q58: Which Cisco feature drops packets with spoofed source addresses?

  • A. ACLs
  • B. uRPF
  • C. DHCP Snooping
  • D. Storm Control

Answer: B. Unicast Reverse Path Forwarding (uRPF).

Q59: Which OSPF area is required in every OSPF network?

  • A. Area 0
  • B. Area 1
  • C. Stub Area
  • D. NSSA

Answer: A. Area 0 (Backbone area).

Q60: Which automation tool uses YAML files for configuration management?

  • A. Chef
  • B. Puppet
  • C. Ansible
  • D. Terraform

Answer: C. Ansible.
