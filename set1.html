Q1: Which protocol is used to automatically assign IP addresses to hosts?

  • A. ARP
  • B. DHCP
  • C. ICMP
  • D. DNS

Answer: B. DHCP – Dynamic Host Configuration Protocol automatically assigns IP addresses.

Q2: A switch receives a frame with an unknown destination MAC address. What does it do?

  • A. Drops the frame
  • B. Floods out all ports except incoming
  • C. Sends to default gateway only
  • D. Stores frame until MAC is learned

Answer: B. Floods out all ports except incoming port.

Q3: What is the default administrative distance for a static route?

  • A. 0
  • B. 1
  • C. 90
  • D. 120

Answer: B. 1 – Static routes have an AD of 1.

Q4: Which layer of the OSI model do routers primarily operate at?

  • A. Data Link
  • B. Transport
  • C. Network
  • D. Physical

Answer: C. Network layer (Layer 3).

Q5: Which protocol maps IP addresses to MAC addresses?

  • A. DNS
  • B. ICMP
  • C. ARP
  • D. DHCP

Answer: C. ARP – Address Resolution Protocol.

Q6: Which IPv6 address type is similar to IPv4 private addresses?

  • A. Link-local
  • B. Global unicast
  • C. Unique local
  • D. Anycast

Answer: C. Unique Local Addresses (ULA).

Q7: Which VLAN is the default native VLAN on Cisco switches?

  • A. VLAN 0
  • B. VLAN 1
  • C. VLAN 100
  • D. VLAN 99

Answer: B. VLAN 1 is default native VLAN.

Q8: What is the default spanning-tree protocol on Cisco switches?

  • A. RSTP (802.1w)
  • B. PVST+
  • C. MST
  • D. STP (802.1D)

Answer: B. PVST+ (Per VLAN Spanning Tree Plus).

Q9: Which command displays the IP routing table on Cisco IOS?

  • A. show run
  • B. show interfaces
  • C. show ip route
  • D. show arp

Answer: C. show ip route.

Q10: Which type of NAT maps multiple inside local addresses to a single inside global address?

  • A. Static NAT
  • B. Dynamic NAT
  • C. PAT (Port Address Translation)
  • D. NAT64

Answer: C. PAT – many-to-one translation using ports.

Q60: Which Cisco technology separates the control plane from the data plane to enable automation?

  • A. SNMP
